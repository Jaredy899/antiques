# Antique Images Directory

This directory is for storing antique images to be displayed on the website.

## Home Page Hero Image

Place your main antique display image here and name it 'antique-display.jpg'.
This image will appear in the hero section of the home page.

Recommended image size: at least 800x600 pixels, landscape orientation works best.

## Category Images

For each category (Furniture, Jewelry, Art, etc.), you can add corresponding images:
- furniture.jpg
- jewelry.jpg
- art.jpg
- collectibles.jpg
- china-glassware.jpg
- home-decor.jpg

These images will be displayed in the Featured Categories section.

## Antique Item Images

When adding specific antique items to the gallery, save their images in this directory
with descriptive names. For example:
- vintage-oak-chair.jpg
- art-deco-lamp.jpg
- silver-pocket-watch.jpg

## Image Format Guidelines

- Use JPG format for photographs (recommended for most antique images)
- Use PNG format for images that need transparency
- Optimize images for web to keep file sizes reasonable
- Maintain consistent aspect ratios for a cohesive look
